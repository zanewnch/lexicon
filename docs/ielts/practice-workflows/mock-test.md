# Mock Test Workflow

## Purpose

Mock Test is for practicing IELTS Listening under exam-like pressure.

The goal is not to understand every sentence. The goal is to answer questions accurately while the audio plays once, just like the real IELTS test.

## Best Materials

Use high-quality test-style materials:

- IELTS official sample tests
- British Council practice tests
- IDP practice tests
- Cambridge IELTS books
- Carefully checked mock tests

Avoid using random online tests for score tracking unless the answer key is reliable.

## When To Practice

Recommended frequency:

- Once per week for a full 40-question listening test.
- Twice per week if the exam is close.
- Avoid doing full mock tests every day unless reviewing mistakes properly.

Use this mode when you want to check:

- Current score level
- Timing control
- Question-type weakness
- Exam stamina
- Whether practice is improving your score

## Setup Before Starting

Prepare:

- Audio file or online audio page
- Question paper
- Answer sheet or digital answer form
- Timer
- Pen and paper, or keyboard input
- Quiet environment

Rules:

- Do not pause the audio.
- Do not replay the audio.
- Do not check the transcript during the test.
- Do not check answers until all sections are finished.

## Full Practice Workflow

### Step 1: Choose One Complete Test

Pick one complete IELTS Listening test with 4 sections and 40 questions.

Record basic information:

- Test source
- Test name
- Date
- Target score

Example:

```text
Source: British Council
Test: Listening Practice Test 1
Date: 2026-05-28
Target: 32/40
```

### Step 2: Preview Questions Before Each Section

Before playing each section, quickly scan the questions.

Look for:

- Word limit
- Question type
- Names
- Numbers
- Dates
- Units
- Map labels
- Options in multiple choice

Do not try to deeply understand everything. The purpose is to know what information to listen for.

### Step 3: Play Audio Once

Start the audio and answer while listening.

During playback:

- Follow the question order.
- Move on if one answer is missed.
- Write a guess if unsure.
- Watch for correction phrases like "actually", "sorry", "no", "rather", and "instead".
- Watch for paraphrases rather than exact question words.

Important rule:

- Never sacrifice the next three answers for one missed answer.

### Step 4: Transfer Or Finalize Answers

After all sections are done, finalize answers.

Check:

- Spelling
- Singular/plural
- Word limit
- Capitalization consistency
- Numbers and units
- Date format

Do not change answers randomly. Only change an answer if there is a clear reason.

### Step 5: Score The Test

Compare with the answer key.

Record:

- Raw score
- Estimated band
- Wrong question numbers
- Question types missed
- Reason for each mistake if obvious

Example:

```text
Score: 29/40
Estimated band: 6.5
Wrong questions: 4, 8, 11, 14, 19, 23, 25, 27, 31, 36, 39
Weak types: multiple choice, map labelling, spelling
```

### Step 6: Review Wrong Answers

For each wrong answer, identify why it was wrong.

Use these labels:

- Did not hear the word
- Heard it but spelling was wrong
- Missed plural/singular
- Chose distractor
- Missed paraphrase
- Lost position in audio
- Did not understand accent
- Did not understand vocabulary
- Wrote too many words
- Changed correct answer to wrong answer

### Step 7: Create Follow-Up Practice

Every mock test should create follow-up work.

Examples:

- If many number mistakes: do a Targeted Drill for numbers and prices.
- If many spelling mistakes: do name/address dictation.
- If multiple choice is weak: review distractor patterns.
- If Part 4 is weak: do Deep Listening on the lecture section.

## Completion Standard

A Mock Test session is complete only when:

- The full test is finished.
- Score is recorded.
- Wrong answers are reviewed.
- Mistake reasons are labeled.
- At least one follow-up drill is planned.

If only the score is checked, the practice is incomplete.
