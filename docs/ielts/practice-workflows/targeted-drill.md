# Targeted Drill Workflow

## Purpose

Targeted Drill is for attacking one weak point at a time.

The goal is not to finish a full IELTS test. The goal is to repeat a specific listening skill until it becomes reliable.

## Best Materials

Use short, focused materials:

- Generated IELTS-style scripts
- Local TTS audio
- Extracted clips from past wrong answers
- Short official sample segments
- Self-made number, date, name, and spelling drills

Generated materials are especially useful here because they can be customized around personal mistakes.

## When To Practice

Use Targeted Drill after:

- Mock Test reveals repeated mistakes.
- Deep Listening identifies a sound pattern problem.
- You notice a weak category, such as numbers or map directions.

Recommended frequency:

- 10-20 minutes per drill.
- 1 weak point per session.
- 3-5 sessions per week.

Avoid mixing too many skills in one drill.

## Drill Categories

High-priority IELTS listening drill targets:

- Dates
- Times
- Prices
- Phone numbers
- Addresses
- Names and spelling
- Singular/plural endings
- Map directions
- Table completion
- Academic vocabulary
- Synonym replacement
- Paraphrase recognition
- Distractor correction
- Multiple-choice traps

## Full Practice Workflow

### Step 1: Choose One Weak Point

Pick exactly one target.

Good examples:

- Dates in Section 1
- Prices and discounts
- Street addresses
- Names with spelling
- Map directions
- Multiple-choice distractors
- Lecture signposting phrases

Bad examples:

- Practice all listening
- Improve Part 3
- Get better at accents

The target must be specific enough to generate repeated examples.

### Step 2: Set A Mini Goal

Define success before starting.

Examples:

```text
Target: prices
Goal: 18/20 prices correct

Target: names and spelling
Goal: 15/20 names spelled correctly

Target: map directions
Goal: label 8/10 places correctly
```

### Step 3: Generate Or Select Drill Material

Create a short drill around the target.

For generated practice, specify:

- IELTS section
- Scenario
- Target skill
- Number of questions
- Difficulty
- Accent preference

Example generation request:

```text
Create a 10-question IELTS Listening Section 1 drill about booking a hotel.
Focus on dates, prices, phone numbers, and spelling names.
Use a natural British English conversation.
Include transcript, questions, and answer key.
```

### Step 4: First Attempt Under Light Pressure

Play the audio once or twice depending on the drill type.

Recommended rules:

- For exam-style drill: play once.
- For skill-building drill: allow two plays.
- For spelling drill: allow pause after each item.
- For map drill: play once first, then replay for review.

Answer all questions before checking.

### Step 5: Check And Label Mistakes

Compare answers with the key.

For each mistake, label it:

- Did not hear
- Heard but spelled wrong
- Confused similar sound
- Missed correction
- Missed unit
- Missed plural
- Chose distractor
- Lost direction
- Misread question

### Step 6: Repeat With Variation

Repeat the same skill with new examples.

Do not only repeat the same audio until memorized. The goal is recognition across variation.

Examples:

- Same target, different scenario
- Same numbers, different speaker speed
- Same map language, different map
- Same spelling task, different names

### Step 7: Promote Or Keep The Weak Point

After the drill, decide:

- Promoted: accuracy is high enough; revisit later.
- Keep: still weak; schedule another drill.
- Convert: problem is deeper and needs Deep Listening.

Example:

```text
Target: prices
Score: 16/20
Decision: Keep
Reason: still confusing fifteen/fifty and thirteen/thirty
Next drill: number contrast drill
```

## Completion Standard

A Targeted Drill session is complete when:

- One specific weak point is practiced.
- A score is recorded.
- Mistakes are labeled.
- A next action is chosen.

Targeted Drill is not complete if the practice is too broad or no weak point is tracked.
